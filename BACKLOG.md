# Feature Ideas Backlog

All features below are designed to be toggle-able. Priority: P1 (high) / P2 (medium) / P3 (low).

## Game Modes
- [ ] P2 - **Infection Mode**: Killed props become hunters
- [ ] P2 - **King of the Props**: Props earn points by staying in a highlighted zone
- [ ] P2 - **2 Hunters Mode**: Fixed 2 hunters regardless of player count
- [ ] P3 - **Timed Swap**: Roles swap mid-round after a timer

## Progression & Economy
- [ ] P2 - **XP/Level System**: Earn XP per match, level up for cosmetic unlocks
- [ ] P2 - **Perks (non-pay-to-win)**: Small passive abilities (e.g., slightly faster reload, quieter footsteps)
- [ ] P3 - **Daily/Weekly Quests**: "Get 5 kills as hunter", "Survive 3 rounds as prop"
- [ ] P3 - **Battle Pass (free)**: Seasonal reward track

## Cosmetics
- [ ] P2 - **Weapon Skins**: Visual-only gun color/pattern changes
- [ ] P2 - **Prop Skins**: Unique visual variants for certain props
- [ ] P3 - **Trails**: Particle trails when moving
- [ ] P3 - **Emotes/Taunts**: Props can taunt with sounds and animations
- [ ] P3 - **Victory Poses**: Shown on round/match end screen

## Competitive
- [ ] P2 - **Ranked Matchmaking (ELO)**: Separate ranked queue with skill-based matching
- [ ] P3 - **Leaderboards**: Global and friends leaderboards
- [ ] P3 - **Seasons**: Reset ranks periodically with rewards

## Social
- [ ] P1 - **Proximity Voice Chat**: WebRTC-based, 3D spatial audio
- [ ] P2 - **Friends List**: Add friends, invite to room
- [ ] P3 - **Spectator Mode**: Watch ongoing matches
- [ ] P3 - **Anti-Stream-Sniping**: Delay spectator view, hide room codes

## Maps & Content
- [ ] P1 - **Map Voting**: Players vote on next map between rounds
- [ ] P2 - **Additional Maps**: Factory, School, Supermarket, Pirate Ship
- [ ] P2 - **Workshop/Custom Map Pipeline**: JSON-based map format, community maps
- [ ] P3 - **Random Events**: Blackout (10s darkness), alarm (noise everywhere), fog, moving crane

## Technical
- [ ] P1 - **AI Bots**: Fill rooms with bot players (basic pathfinding + behavior)
- [ ] P2 - **Replay System**: Record match snapshots, playback
- [ ] P2 - **WebGL/Mobile Build**: Responsive UI, touch controls
- [ ] P3 - **Server Regions**: Multiple server deployments with auto-region selection
- [ ] P3 - **Host Migration**: Seamless host transfer on disconnect (if not using dedicated)

## Accessibility
- [ ] P1 - **Full Key Remapping**: Customizable keybinds for all actions
- [ ] P1 - **Colorblind Modes**: Deuteranopia, Protanopia, Tritanopia filters
- [ ] P2 - **Subtitles/Sound Visualization**: Visual indicators for directional sounds
- [ ] P2 - **UI Scaling**: Adjustable HUD size
- [ ] P3 - **Screen Reader Support**: ARIA labels for menus

## Quality of Life
- [ ] P2 - **Killcam**: Short replay of death from killer's perspective
- [ ] P2 - **Match History**: View past match results
- [ ] P3 - **Player Reports**: Report system for toxic behavior
- [ ] P3 - **Auto-kick AFK**: Remove inactive players after timeout
